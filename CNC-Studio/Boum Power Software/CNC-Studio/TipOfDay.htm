Chargement automatique du SUB

Dans le programme MAIN, insérer une ligne avec le contenu suivant :

(<SUB=NomFichier>)

Lors de l'ouverture du programme MAIN, CNC-Studio ouvrira automatiquement le programme SUB désigné par NomFichier.

Commentaire fonctionnel

Utilisez les commentaires fonctionnels dans vos programme : l'éditeur de CNC-Studio récupère ces informations pour améliorer la présentation de vos programmes à la conception ainsi que pour l'impression.

Le caractère suivant la parenthèse ouvrante détermine la fonction du commentaire. Les caractères suivants sont réservés :

  • _ pour les titres : (_Fraisage) sera récupéré comme titre principal dans les navigateurs.
  • * pour les sous-titres : (*Mèche) sera récupéré comme sous-titre.
  • ! pour les mises en garde : (!Arrosage) attirera votre attention à la lecture du programme.
  • < pour les macros : (<MACRO>) voire l'utilisation des macros.

Ajoutez vos propres astuces

Vous avez certainement pu apprécier les astuces du jour.

CNC-Studio vous permet par le biais de l'onglet "Nouvelle astuce" d'ajouter vos propres astuces afin d'en faire bénéficier les autres utilisateurs (sous réserve que le fichier des astuces soit partagé).

  • de nouveaux procédés d'usinage
  • ou de tout autre domaine relatif à l'utilisation de machines CNC
