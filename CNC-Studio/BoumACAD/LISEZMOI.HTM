Installation

Les modifications suivantes sont à apporter au fichier Acad.lsp (noter que le nom de ce fichier peut varier d'une version à l'autre d'AutoCAD : AcadR13.lsp AcLT.lsp, etc) se référer à la documentation d'AutoCAD pour plus de détails.

Ajouter la ligne suivante :

(load "BoumACAD")

Créer ou modifier la fonction S::STARTUP afin d'appeler (BoumAcadStartup)

(defun S::STARTUP()
(BoumAcadStartup)
(princ)

)

 

Veiller également aux réglages d'AutoCAD: le chemin d'accès au répertoire d'installation de ce logiciel doit figurer dans :

Options/Préférences/Environnement/Support (AutoCAD R13)

ou dans le

chemin de recherche de fichiers de support (AutoACAD R14 ?, AutoCAD LT)
