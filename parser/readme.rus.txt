testhtmlparser.php - пример использования
rebuildgrammar.php - компилятор грамматики
htmlgrammar.dat - грамматика
htmlgrammar.cmp - откомпилированная грамматика
test.html - файл для теста html parsera
common.inc - вспомогательные функции
imgsrcchange.inc - пример подмены src значения параметра тега img

Установка проста. Копируете все файлы в какой-нть каталог на свой вебсервер и
далее по ссылочке вызываете http://..../testhtmlparser.php
Результатом будет являться стартовая страница PHPClub которая лежит в test.html.
Она формируется по дереву тегов, полученных в результате парсинга,
а после нее распечатанное дерево тегов.
